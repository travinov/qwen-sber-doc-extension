# Gigacode / Gemini Extension Context

Используйте это расширение, когда требуется задокументировать Python-модуль или директорию проекта в официальном русскоязычном стиле со строгой структурой.

Важно:
1. Для Gigacode/Gemini MCP-сервер `gigadoc-mcp` обычно подключается отдельно через `/mcp` или файл настроек клиента.
2. Команды `/doc:sber` и `/sber` рассчитаны на доступность MCP tools:
   - `analyze_python_target`
   - `build_sber_doc_outline`
   - `build_sber_project_outline`
   - `validate_sber_doc`

Правила применения:
1. Для редакционного стиля и структуры сначала ориентируйтесь на skill `sber-doc-style`.
2. Для документирования конкретного файла или директории используйте команду `/doc:sber <path>` или `/sber <path>`.
3. Не смешивайте официальный стиль с разговорной подачей.
4. Не придумывайте параметры, поведение и примеры. Если данные не подтверждены кодом, явно помечайте ограничение.
